PREDATIX - VERSION REPARADA PARA GITHUB PAGES

IMPORTANTE:
- Esta version tiene comentarios de ayuda, pero NO rompen las imagenes.
- Para editar textos, busca: EDITAR AQUI
- Para editar imagenes, busca: CAMBIAR IMAGEN AQUI

ESTRUCTURA:
- index.html
- curso-de-0-a-ds.html
- css/styles.css
- js/script.js
- assets/

COMO CAMBIAR IMAGENES:
1. Sube tu imagen a la carpeta assets/
2. Cambia la ruta dentro de src="..."
Ejemplo:
src="assets/mi-nueva-imagen.png"

COMO PUBLICAR EN GITHUB:
1. Descomprime este ZIP.
2. Sube la carpeta predatix_web completa a un repositorio.
3. En GitHub ve a Settings > Pages.
4. En Branch elige main y carpeta /root.
5. Guarda y espera el link de GitHub Pages.

NO BORRAR:
- css/
- js/
- assets/
- .nojekyll
