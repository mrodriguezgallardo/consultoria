# Predatix Landing Page

Sitio web para Predatix con página principal y landing de curso.

## Estructura

```txt
predatix_web/
  index.html
  curso-de-0-a-ds.html
  css/
    styles.css
  js/
    script.js
  assets/
    predatix_white.png
    predatix_original_transparent.png
    predatix_favicon_symbol.ico
```

## Páginas

- `index.html`: sitio principal de Predatix.
- `curso-de-0-a-ds.html`: landing de venta del curso "De 0 a Científico de Datos".

## Imágenes

La landing del curso usa imágenes con URL de Unsplash para que puedas cambiarlas después.

## Uso

Sube la carpeta completa a GitHub y activa GitHub Pages.
